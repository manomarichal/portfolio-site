Personal Projects


Indie games I am developing on my own and hope to release one day.

Bagels of War


I have been designing and developing the game Bagels of War for over a year now, with the help of some artist and sound designer friends. It is my first large project that made it out of development hell, and I hope to publish it on Steam later this year.

Currently, there is a demo featuring 5 levels, with most mechanics and systems in place. I have programmed every little detail myself, so I gained many insights to the different areas of game development, from data storage to user interface and gameplay programming.


Gregs Dream


Gregs dream is a metroidvania that I have worked on with an artist/sound designer friend for over a year. It is about a young boy named Greg who gets imprisoned in a nightmare and has to find a way to escape. It is inspired by classic zelda dungeons and other metroidvanias like Hollow Knight. I do the game and level design, and the programming.

The game currently features over two hours of gameplay, but still has a long way to go. It was our first attempt at a bigger, marketable game, but fell prey to scope creep and overambition, hence it currenly lays dormant in my pile of unfinished projects



Size--


A tool made by me and two friends to compare different compression algorithms. We implemented six algorithms to convert data into context-free grammars, which are stored in a data format we came up with, called PAL. The data is then converted to binary strings using huffman encoding, to further reduce file size. The tool comes with a complete CLI, and tools to visualize and compare the performance of the algorithms.

C compiler


Software to compile C code to LLVM and/or to MIPS Assembly. It is able to compile most of the C functionality, including flow control and functions. Furthermore, it also has semantic error detection, informing you of any semantic erros found in your code and where. You can find the full list of what it does and does not support on the Github page.
